
# YouTube Downloader با GitHub Actions

این پروژه به کمک GitHub Actions ویدیوهای یوتیوب را دانلود می‌کند.

## نحوه استفاده
1. به تب **Actions** برو.
2. روی **YouTube Downloader** کلیک کن.
3. **Run workflow** را بزن و لینک یوتیوب را وارد کن.
4. بعد از اتمام، ویدیو را در پوشه `videos` پیدا کن.

## نیازمندی‌ها
فقط یک اکانت گیت‌هاب کافیست.
